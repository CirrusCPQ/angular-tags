angular-tags

Pure AngularJS tagging widget with typeahead support courtesy of ui-bootstrap

D/L .zip D/L .tar.gz View on GitHub

What's angular-tags?

angular-tags is a directive for AngularJS written in pure AngularJS which allows you to enter "tags" into an input box. The only prerequisite is the ui.bootstrap.typeahead module from the UI Bootstrap project, for typeahead support.

How does it work?

Like this:

And the markup: 

<tags 
  data-options="{addable: true, classes: {white: 'black', red: 'red'}}"
  data-src="s as s.name for s in sourceTags"
  data-model="tags"></tags>

Sweet, how do I get it? 

bower install angular-tags
Grab Bower and install this way, or just download the archive or clone from GitHub.
